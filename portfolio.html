    Montador de vilas do jogo Clash of Clans

    Desenvolvido na linguagem Java com auxílio do widget toolkit Swing.

    Clash of Clans é um jogo dos jogos mais famosos de estratégia feito para mobile, desenvolvido pela Supercell, possui mais de 100 milhões de downloads na Google Store.

    O principal motivo para criação desse projeto foi para facilitar a criação de novos layouts de defesa para o jogo no PC.

    O projeto possui:

  • Menu de criação de novos layouts;
  • Opção de salvar e carregar layouts;
  • Botão de ajuda com comandos chave.

    Jogo Tetris

    Desenvolvido na linguagem Lua com auxílio de Corona SDK.

    Esse foi o meu primeiro jogo criado para Android. Ele possui as funcionalidades básicas do Tetris e algumas implentações extras:

  • Modo Endless: O jogo termina apenas quando não tiver espaço para a próxima peça;
  • Modo BIG: Endless e peças de tamanho reduzido pela metade;
  • Modo Clear: O jogo termina quando não houver mais blocos-chave no grid;
  • Level: Numero de linhas iniciais geradas aleatoriamente;
  • Speed: Velocidade que o bloco desce.

    Jogo 2D estilo Defense com física

    Desenvolvido na linguagem C# com auxílio da engine Unity.

    Inspirado no jogo Clash of the Olympians, foi o meu primeiro jogo utilizando a engine Unity. Trata-se de um jogo de ação com alguns elementos de RPG. A idéia principal desse jogo é defender a torre controlando um personagem do jogo onde é possível atirar flechas nos inimigos.

    Esse jogo possui as seguintes funcionalidades:

  • Sistema de vida e energia;
  • Física controlada pelo Unity;
  • Mira para ajudar na jogabilidade;
  • Sistema de nível e habilidades;
  • Sistema de efeitos para flechas especiais;
  • Gerador de inimigos;
  • GUI de estado do personagem e inimigos.
  • Sistema básico de salvar o level do personagem.
    •

    Os próximos passos desse jogo são:

  • Implementar de fases do jogo;
  • Criação de uma nova interface para transição dessas fases;
  • Melhoria no sistema de salvar o estado do jogo

    Jogo 2D estilo Defense/RPG

    Desenvolvido na linguagem C# com auxílio da engine Unity.

    Inspirado no sistema de rotação de habilidades do jogo World of Warcraft, é o meu segundo jogo utilizando a engine Unity. Trata-se também de um jogo de ação com alguns elementos de RPG. O objetivo desse jogo é eliminar os inimigos utilizando habilidades do personagem controlada pelo jogador.

    Esse jogo possui as seguintes funcionalidades:

  • Sistema de vida e energia;
  • Sistema de nível e habilidades;
  • Gerador de inimigos;
  • Sistema de equipamentos e inventário;
  • Sistema de efeitos das habilidades;
  • GUI de estado do personagem e inimigos;
  • Sistema de salvar e carregar o estado do jogo;
  • Transição de cenas sem perder dados.
    •

    Os próximos passos desse jogo são:

  • Melhorar o sistema de equipamento;
  • Implementar fases do jogo;
  • Melhorar sistema de salvar e carregar o estado do jogo
